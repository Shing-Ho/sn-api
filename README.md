# SimpleNight API

## Building

### Prerequisites
- python3.7 (preferred with [pyenv](docs/pyenv.md))

## Getting Started
After pre-requisites are satisfied, from root of directory, run:
```bash
python3.7 -m venv .venv
. .venv/bin/activate
pip install --upgrade pip

pip install -r requirements-dev.txt

pre-commit install
```
